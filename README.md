**01 · Dark Mode Toggle**

Switches the entire page between light and dark themes
Uses data-theme attribute on <html> and CSS custom properties
Concepts: classList.toggle(), attribute mutation, CSS variables

**02 · Click Counter**

Increment, decrement, and reset a number
Color changes: green for positive, red for negative
Bump animation on every click
Concepts: State variables, event listeners, DOM text updates

**03 · Live Writer**

Real-time character and word count as you type
Animated progress bar (turns amber at 75%, red at 100%)
Clear button resets everything
Concepts: input event, string methods, dynamic class toggling

**04 · Task Manager**

Add tasks via button click or pressing Enter
Check off completed tasks with a strikethrough effect
Delete individual tasks
Live completion stats (e.g. "2 of 5 tasks complete")
Concepts: createElement(), appendChild(), array manipulation

**05 · FAQ Accordion**

Click any question to expand the answer
Only one panel open at a time
Smooth CSS transition on open/close
Concepts: classList.toggle(), max-height CSS animation, single-open logic

**06 · Color Theme Picker**

6 accent color swatches to choose from
Updates the entire site's accent color instantly
Live preview panel shows selected color name and hex
Concepts: CSS custom properties (setProperty()), dynamic styling
Bonus · DOM Mutation Tracker
A live bar below the hero that pulses on every DOM change
Logs a description of every mutation (e.g. counter.textContent → 5)
Tracks total interactions and mutations in the hero stats
Concepts: Real-time feedback, reflow trick for restarting CSS animations
